
# Python Sets

## Overview

This folder contains my notes, examples, practice questions, and experiments on **Python Sets** as part of my Python learning journey.

A **set** is an unordered, mutable collection of unique elements. Sets are commonly used to remove duplicate values and perform mathematical set operations such as union, intersection, and difference.

## Key Learnings

- Sets store only unique values.
- Sets are unordered and cannot be accessed using indexes.
- Sets are mutable, so items can be added or removed.
- Sets are useful for removing duplicates from data.
- Set operations make comparing datasets simple and efficient.
