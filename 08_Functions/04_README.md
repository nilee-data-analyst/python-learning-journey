##  Concepts Learned

### Functions
Functions allow you to write reusable blocks of code that perform a specific task.

### Parameters
Parameters let you pass data into a function.

### Return
The `return` statement sends a value back to the caller.

### Default Arguments
Provide a default value when no argument is passed.

### Keyword Arguments
Pass arguments using parameter names, making the order flexible.

### *args
Accepts any number of positional arguments as a tuple.

### **kwargs
Accepts any number of keyword arguments as a dictionary.

### Lambda Functions
Small anonymous functions commonly used for sorting and data transformations.

### Scope
- Local variables exist only inside a function.
- Global variables can be accessed throughout the program.
